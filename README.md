# LeetMetric

LeetMetric is a web-based tool that allows users to track and visualize their LeetCode problem-solving progress in an interactive and user-friendly way.

It fetches real-time data using the LeetCode GraphQL API and displays insights such as total solved problems, difficulty-wise breakdown, and submission statistics.

---

## Features

- Search any LeetCode username  
- View total problems solved  
- Difficulty-wise stats (Easy, Medium, Hard)  
- Submission statistics displayed in cards  
- Circular progress indicators  
- Clean and responsive UI  

---

## Tech Stack

- HTML  
- CSS  
- JavaScript  
- LeetCode GraphQL API  

---

## How It Works

1. Enter a LeetCode username  
2. Click on the search button  
3. The app fetches data using the GraphQL API  
4. Progress and statistics are displayed visually
